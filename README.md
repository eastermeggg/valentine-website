# Valentine's Day - J-Bae Edition

A fun, playful, pixel-art themed Valentine's Day website.

## Features

- Pixel art heart animation
- Floating hearts and twinkling stars
- Click-to-explode hearts button with retro sound effects
- Love power meter (it's over 9000!)
- Scroll-reveal "Why you're the best" section
- Konami code secret message (up up down down left right left right B A)
- Fully responsive
- Pure HTML/CSS/JS - no dependencies

## Deploy

Static site - just serve `index.html`. Works on Vercel, Netlify, GitHub Pages, or any static host.
